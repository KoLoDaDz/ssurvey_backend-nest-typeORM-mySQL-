всі контролери при зверненні через get метод без закінчення(приклад:http://localhost:3000/answer-one) повертають масив який зберігає json інформацю про кожен рядок у таблиці
для додавання в табицю потрібно робити post запит(приклад:http://localhost:3000/answer-one) в якому мають передаватися данні описані у dtos/dto.ts файлі

при зверненні через метод delete потрібно звертатися до до конкретного id в базі(приклад:http://localhost:3000/answer-one/3), даним методом можна видалити кокретний рядок з таблиці
